# テトリス

ブラウザで動作するシンプルなテトリスゲームです。

## デモ

[Vercel でホスティングされたデモを見る](https://tetris-game-8uhy096gl-hiros-projects-76a84dd8.vercel.app)

## 機能

- 7種類のテトリミノ（I、O、T、S、Z、J、L）
- スコア、レベル、ライン数の表示
- レベルアップによる落下速度の増加
- 一時停止機能
- ゲームオーバー判定とリスタート機能

## 操作方法

- **← →** : 左右移動
- **↓** : 高速落下
- **↑** : 回転
- **Space** : 一時停止

## ローカルでの実行

1. リポジトリをクローン
```bash
git clone [your-repository-url]
cd tetris
```

2. `index.html` をブラウザで開く

## 技術スタック

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
